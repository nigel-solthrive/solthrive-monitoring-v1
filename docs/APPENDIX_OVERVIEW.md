# SolThrive V1 – Documentation & Appendices

This folder contains the main engineering spec PDF and supporting notes.

## Files

- `SolThrive_V1_FullSpec_Manual_Simple.pdf`  
  Simple black‑and‑white engineering specification for V1, including:
  - System purpose
  - Architecture
  - Hardware & software overview
  - Appendices A–H (glossary, CT guide, Modbus primer, Pi watchdog, wiring, bench/field checklists, troubleshooting)

- `APPENDIX_OVERVIEW.md` (this file)  
  Quick text summary of what’s in the PDF and how to use it.

## How to Use

1. **Start with the PDF** to understand the full system and constraints.
2. Use the **appendix sections** as checklists when:
   - Selecting CTs and meter
   - Wiring the panel
   - Running bench tests
   - Doing on‑site installation
3. Update this folder over time with:
   - New versions of the spec
   - Photos from real installs
   - Any revised procedures.
